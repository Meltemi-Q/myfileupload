# MyFileUpload

Secure file upload server with Bento UI, theme switching, and i18n (zh/en).

## Features
- 🔒 Password-protected upload
- 🎨 Bento UI with dark/light theme
- 🌍 Bilingual (Chinese/English)
- 📤 Drag & drop file upload
- 📋 File list with download links

## Quick Start

```bash
node file-upload-server.mjs
```

Server runs on port 47891 by default.
